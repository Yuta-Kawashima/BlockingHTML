# BlockingHTML
HTMLとCSSをより深く理解できるようにブロック構造でHTML要素を表示しました。

style2.cssファイル内のborderプロパティを消していただいて、任意の写真やリンクなどを貼り付けていただければ、デザイン次第で最近流行りのWebページは簡単に作成できるのではと愚行いたします。

基本的にはProgateにおけるHTML&CSSの中級程度の知識があれば問題なく理解可能であると考えています。
